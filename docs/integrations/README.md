# Integrations

AgentMemDB integrates with popular agent frameworks and vector stores.

| Integration | Use Case | Doc |
|-------------|----------|-----|
| [LangChain](langchain.md) | VectorStore for RAG, retrieval chains | `AgentMemDBVectorStore` |
| [LangGraph](langgraph.md) | BaseStore for long-term memory | `AgentMemDBStore` |

## Quick Links

- **LangChain:** `make langchain-example`
- **LangGraph:** `make langgraph-example`, `make langgraph-agent`
